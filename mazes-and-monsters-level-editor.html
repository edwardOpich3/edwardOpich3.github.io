Mazes & Monsters Level Editor

While developing Mazes and Monsters, I eventually tired of manually creating the maps via text editor, so I created a very basic tool to make it a little bit easier to tell just what it is I was doing.

The level editor was made in C++, using the Allegro library for graphics.
