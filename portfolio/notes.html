Notes

All images, scripts, and the site's unique CSS was created by me. The font used in the header is 'VT323', found on Google Fonts.

The CSS Framework is Skeleton. The changes I made include changing the base font, changing the default colors on links, and changing the default color on the font.

My Above and beyond work was coding the navigation buttons from scratch. It's not anything super impressive, but I figured it'd be better to play it safe than to try for something really cool and not be able to finish it in time.

Honestly, there weren't too many challenges encountered while working on this. Compared to the previous project, Skeleton made creating the site significantly easier; perhaps that's the sign of a good framework! The biggest challenge was, honestly, finding things to change in the framework. Skeleton has its name for a reason; it's a skeleton, there's not much there to change! So I had to make do with that.

I'm sure there are plenty of ways to improve the site; someone with more of an eye for design might think up a better color palette, or add some flourishes to the CSS to really make the site pop, like some shadows or some strategically placed divs and rounded edges. But through and through, I'm pretty happy with how it turned out!
