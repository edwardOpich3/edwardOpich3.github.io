Drip Game

You're a drip of paint, sliding down a wall. You don't know where you started. You don't know why. You don't know when you'll stop. You do know, however, that you don't want to stop. Slide as far as you can down different walls, dodging obstacles, and grabbing powerups all the while!

Drip Game is a game I started on my own in 2017, and has since ceased development after having lost interest.

The game was my first attempt at a game targeted to mobile devices, and was eventually meant to test out the Allegro library's mobile capabilities.

The game was developed using the Allegro library, in C++.
